# Prior Art

This document catalogs existing standards, specifications, and licenses that are related to the AI License Notice, and explains how this work differs from them.

## SPDX License List

[spdx.org/licenses](https://spdx.org/licenses/)

SPDX maintains a comprehensive list of software licenses. It does not currently include AI specific license identifiers or provenance declaration standards.

## REUSE Specification

[reuse.software](https://reuse.software/)

REUSE is a specification for declaring copyright and licensing information in source code files. It focuses on per file license identification. It does not address AI authorship or provenance.

## OpenRAIL and BigCode Licenses

OpenRAIL and BigCode licenses govern the use of AI models, not code generated with AI assistance. They apply to model weights and outputs derived from running the model, which is a different concern than code authored with AI tool assistance in a developer IDE.

## Difference

The AI License Notice:

- Applies to code (not models)
- Is a notice (not a license)
- Attaches to existing licenses
- Is informational (no legal terms)
- Is machine readable
- Works with any governance engine

These distinctions make this work complementary to, not a replacement for, existing standards.
